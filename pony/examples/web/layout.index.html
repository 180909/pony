Пример автоматического создания колонок

Эта программа демонстрирует возможности автоматически генерируемой разметки страницы.

Установка

Для того, чтобы этот пример отображался нормально, необходимо установить две дополнительные библиотеки:

  • libxml2
  • lxml

Скомпилированные дистрибутивы этих библиотек для Windows доступны здесь и здесь.

Описание

Разметка производится с помощью специальных тегов <header>, <footer> и <sidebar>. Эти теги не передаются непосредственно в браузер пользователя, а заменяются набором других тегов, в результате чего содержимое тега <header> выводится в самом верху страницы, а содержимое тега <footer> — в самом низу. Содержимое тега <sidebar> представляет собой боковую колонку документа, и может быть выведено либо слева, либо справа.

Дополнительная информация и примеры содержатся на страницах, перечисленных в боковой колонке.

Компоненты страницы
===================


Для разработчика компонент страницы выглядит как функция, которая может
вызываться из шаблона. Во время генерации страницы компонент разворачивается
в HTML код. Компонент может содержать стили CSS, а также JavaScript.

Пони содержит несколько готовых для использования компонентов.
Разработчик также может создать компоненты сам.
В этой главе речь пойдет о том как применять и создавать компоненты.

Рассмотрим пример применения компонента.




Часто бывает, что фрагмент HTML, оформленный в виде компонента использует
какие то стили CSS или javascript код.
При добавлении компонента на страницу, ссылка на javascript и CSS 
встраивается в заголовок страницы, тогда как сам компонент - в том месте, где его
поместил разработчик. Если разработчик использует один и тот же компонент
несколько раз на странице, ссылки на javascript и CSS будут добавлены только один раз.


def component(old_func, css=None, js=None)
это декоратор с параметрами
может быть список цсс и джскр


например
@component(css='/pony/static/css/rounded-corners.css')
def rounded(markup, **attrs)
